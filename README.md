# IC GERALD REPAIR

Проект представляет собой одностраничный лендинг для строительно-ремонтной компании IC GERALD REPAIR. Сайт разработан с фокусом на адаптивность, семантическую верстку и чистоту кода.

## Технологический стек
* HTML5 (Семантическая верстка)
* CSS3 (Flexbox, Grid, CSS Variables)
* JavaScript (ES6+)

## Технические особенности

В процессе разработки особое внимание уделялось качеству кода и его дальнейшей поддержке:

* **Использование SOLID в JS:** Логика интерактивных элементов реализована с помощью классов JavaScript. Это позволило создать модульную структуру, упрощающую отладку и расширение функционала в будущем.
* **Принцип DRY в CSS:** Активное внедрение CSS-переменных позволило централизовать управление стилями (цвета, отступы). Теперь изменение глобальных параметров дизайна происходит в одном месте, что исключает дублирование кода.
* **Семантическая оптимизация:** Проведен рефакторинг структуры документа. Дублирующий тег `<header>` в нижней части страницы был заменен на семантически корректный `<footer>`, что улучшает доступность сайта и валидность кода по стандартам W3C.
